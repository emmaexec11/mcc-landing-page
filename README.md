## 🚀 Development Workflow

### Local Development:
1. Open `mcc-landing-page.code-workspace` in VS Code
2. Use Live Server extension for local preview
3. Edit `index.html` for changes

### Deployment:
1. Commit changes: `git add . && git commit -m "Description"`
2. Push to GitHub: `git push`
3. Live site updates automatically via GitHub Pages
